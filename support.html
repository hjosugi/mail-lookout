Support

Help and contact information for the mail-lookout Outlook add-in.

Contact

For questions, bug reports, or feature requests, open an issue on GitHub Issues.

Before Reporting

  • Use new Outlook or Outlook on the web.
  • Confirm that the add-in was installed from the latest manifest.
  • Include the Outlook client, browser, and a short description of the problem.

Known Scope

mail-lookout is designed to help review recipients, attachments, and message content before sending. It is not a guarantee that every sending mistake will be prevented.
